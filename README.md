# Consolidaci-n-Inteligente
📊 Consolidador de Ventas (CSV/Excel) Script en Python que automatiza la unión de reportes mensuales.  Capacidades:  Lectura híbrida de archivos .csv y .xlsx.  Limpieza de duplicados y errores de formato.  Generación de resumen ejecutivo en Excel.  Ideal para reducir errores manuales en procesos administrativos y comerciales.
